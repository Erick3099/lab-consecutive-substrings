function consecutiveSubstrings(string) {
  const result = []; 
  for (let x = 0; x < string.length; x++) { 
    // let temp =""
    //  for (let y = x; y < string.length; y++) {
    //   temp += string[y];

    for (let y = x + 1; y <= string.length; y++) {

    }
  }
  return result;

 The task is to generate all consecutive substring from a specific string where the substring contain adjacent character from the original string and be returned in the order they apper.
it uses two nested loops to navigate through the string where the outer one selects the starting index ,while the inner loop selects the ending index.
 Explanation.

 An empty array is created where the results are stored. The outer loop moves through each character in the string and treats it as the starting position of a substring
 while the inner loop moves forward from that position to the end position resulting to consecutive substrings where generated substring are added to the final array in their specific order.
 afterwards the array is returned.,

  TIME COMPLEXITY
  O(n^2) where n is the length of the input string
